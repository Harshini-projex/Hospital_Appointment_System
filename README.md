# CareFlow - Hospital Appointment Management System

CareFlow is a desktop-based medical administration portal designed to streamline patient registration, track clinical statistics, and manage appointment bookings. Built using **Java Swing** for the frontend interface and **MySQL via JDBC** for relational data persistence, the application strictly adheres to modular N-tier software architecture and Object-Oriented Programming (OOP) principles.

## 🚀 Key Features

* **Patient Administration:** Seamlessly register new patients, update records, and safely remove records with full dynamic verification checks.
  
* **Appointment Scheduling:** Book, view, and cancel patient consultation slots across designated doctors and time segments.
  
* **Live KPI Dashboard:** High-fidelity metric cards summarizing total registrations, pending slots, and canceled appointments in real-time.
  
* **Modern UI Styling:** Fully styled using **FlatLaf** for a premium dark layout, optimized column padding, and grid responsiveness.
